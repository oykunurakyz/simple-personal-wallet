# simple-personal-wallet
A simple Python wallet tracker for daily expenses. This is my first Python project that manages daily finances.

##  Features
- **Income & Expense Tracking:** Add your transactions easily.
- **Auto-Save:** All data is saved in `history.txt`.
- **History View:** You can see the date, category, and amount of all past transactions.
- **Smart Balance:** Calculates your total balance automatically.

## Technologies
- **Language:** Python 3.9
- **Modules:** `datetime` (for timestamping)
- **File System:** Text-based data storage

##  How to Use
1. Download `wallet.py`.
2. Run it in your terminal using: `python wallet.py`
3. Follow the menu to manage your money!
